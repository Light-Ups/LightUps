About the Project

LightUps is a high-performance Windows Service and C++/Qt GUI application. It monitors your Uninterruptible Power Supply (UPS) via standard Windows power APIs, ensuring critical event logging and automated system safety during power fluctuations.

Development Environment

The project is built using a specific toolchain to ensure stability and compatibility:

FrameworkQt 6.9.1 (MinGW 64-bit)
IDEQt Creator 18.0.1
CompilerMinGW 13.1.0 (64-bit)
Build SystemCMake 3.30.5 & Ninja 1.12.1
InstallerQt Installer Framework 4.10

Deployment Configuration

To automate the creation of the LightUps installer, follow these steps in Qt Creator:

  • 1. Select Project
    In the left sidebar, click on Projects and ensure LightUps is the active project.
  • 2. Choose Kit
    Under Build & Run, select the correct kit: Desktop Qt 6.9.1 MinGW 64-bit.
  • 3. Deploy Settings
    Open the Deploy Settings tab at the top.
  • 4. Configure Deployment Configuration
    Ensure Active deployment configuration is set to Deploy Configuration, then add these steps:
    • 4a. Build Step: Add a deploy step for build target all.
    • 4b. Install Step: Add a deploy step for install.
    • 4c. Custom Process Step: Add a step with the following details:
      Command: C:\Qt\Tools\QtInstallerFramework\4.10\bin\binarycreator.exe
      Arguments: --offline-only -c config\config.xml -p packages lightups.exe
      Working Directory: %{ActiveProject:NativePath}\installer

Key Features

  • Windows Service: Runs in the background without needing a user login.
  • Event Logging: Professional integration with the Windows Event Viewer.
  • Qt 6.9 UI: Clean and responsive real-time status monitoring.
