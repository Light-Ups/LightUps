About the Project

LightUps is a dedicated Windows Service and GUI application designed to monitor your Uninterruptible Power Supply (UPS) status. It logs events to the Windows Event Log and ensures your system stays safe during power outages.

Features

  • Windows Service: Runs in the background without needing a user to be logged in.
  • Event Logging: Integrated with the Windows Event Viewer for professional logging.
  • GUI Monitor: A clean interface to see real-time status.
  • Auto-Update: Built-in support for updates via the Maintenance Tool.
