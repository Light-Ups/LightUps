About the Project

LightUps is a dedicated Windows Service and GUI application designed to monitor your Uninterruptible Power Supply (UPS) status. It logs events to the Windows Event Log and ensures your system stays safe during power outages.

Deployment Configuration (Qt 6.9)

To build the installer for LightUps using the Qt Installer Framework, follow these steps in the Projects > Deploy Settings tab of Qt Creator:

  1. CMake Install Step: Ensure the install target is selected. This ensures that all binaries and assets are moved to the correct staging directory before the installer is created.
  2. Custom Process Step: Add a new "Custom Process Step" to invoke the binarycreator.exe.
  3. Command: Point this to your Qt Installer Framework path:
    C:\Qt\Tools\QtInstallerFramework\4.10\bin\binarycreator.exe
  4. Arguments: Use the following arguments to create an offline installer:
    --offline-only -c config\config.xml -p packages lightups.exe
  5. Working Directory: Set this to your installer source folder:
    %{ActiveProject:NativePath}\installer

This configuration automates the creation of the lightups.exe installer every time you trigger a "Deploy" within Qt Creator.

Features

  • Windows Service: Runs in the background without needing a user to be logged in.
  • Event Logging: Integrated with the Windows Event Viewer for professional logging.
  • GUI Monitor: A clean interface to see real-time status.
  • Auto-Update: Built-in support for updates via the Maintenance Tool.
